# AUREL — Luxury Candle Brand Identity

AUREL is a luxury candle brand identity project inspired by calm living, soft textures, warm lighting, and timeless minimalism.  
The project explores how visual simplicity and tactile packaging can create a premium and emotionally refined brand experience.

---

## Project Overview

The goal of AUREL was to design a sophisticated candle brand that feels warm, elegant, minimal, and intentional.  
The visual direction focuses on:
- neutral earthy tones
- embossed textures
- soft editorial photography
- premium packaging systems
- modern luxury aesthetics

The entire identity was crafted to communicate calmness, warmth, and timeless living.
